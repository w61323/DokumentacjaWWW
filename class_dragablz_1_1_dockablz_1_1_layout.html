Creates a split in a layout, at the location of a specified TabablzControl.

Parameters
tabablzControlTab control to be split.
orientationDirection of split.
makeSecondSet to true to make the current tab control push into the right hand or bottom of the split.
firstItemProportionSets the proportion of the first tab control, with 0.5 being 50% of available space.

The tab control to be split must be hosted in a layout control. Layout.BranchTemplate should be set (typically via XAML).
