DragablzItem templates contain a thumb, which is used to drag the item around. For most scenarios this is fine, but by setting this flag to trueyou can define a custom thumb in your content, without having to override the template. This can be useful if you have extra content; such as a custom button that you want the user to be able to interact with (as usually the default thumb will handle mouse interaction).
