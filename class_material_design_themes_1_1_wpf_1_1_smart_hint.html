Detailed Description

A control that implement placeholder behavior. Can work as a simple placeholder either as a floating hint, see UseFloating property.

To set a target control you should set the HintProxy property. Use the HintProxyFabricConverter.Instance converter which converts a control into the IHintProxy interface.
